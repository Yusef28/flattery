Error 404

So let me explain. If you try to access a page that doesn't exist on the server we call that an error 404.

Y'know, people don't like to going depth about these things but I think it's interesting to discuss: The first 4 means "client error" so it means the client (you) have don't something wrong like typing in a misspelled url or an url for a page that doesn't exist.

The second number, "0", means "general syntax error" which points again to some misspelling or missing symbols.

The last 4 is almost like a catch all summary. "It's a 4-0-something type error".
